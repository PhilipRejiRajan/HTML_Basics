Introduction

HTML is basically the structure of a webpage.
It consists of a series of elements that tell the web-browser how to display content.
Let's look at some HTML elements, attributes and tags.

Basic HTML Template

Explanation of the above image:
Every HTML page begins with a !DOCTYPE html declaration.
All element tags are enclosed in angle brackets '<' '>'.
The html element is the root element of any HTML page.
The head element contains information about the HTML page.
The title element specifies the title of the webpage.
The body element contains all visible content such as headings, paragraphs, images, links, etc.
The h1 element and p element define a heading and paragraph respectively.
