HTML Attributes

Introduction

All HTML elements can have attributes.
Attributes provide additional information about elements.
Attributes are always specified in the start tag.
Attributes usually come in name/value pairs like: name="value"
Let's look at some HTML attributes.

href attribute:

The a tag defines a hyperlink.
The href attribute specifies the URL of the linked page.

src attribute:

The img tag is used to embed an image in a HTML page.
The src attribute specifies the path to the image to be displayed.

width & height attribute:

The img tag should also contain the width & height attributes, which specify the size of the image(in pixels).

alt attribute:

The alt attribute for the img tag specifies alternate text for an image, if the image for some reason cannot be displayed.

style attribute:

The style attribute is used to add styles to an element, such as color, font, size, etc.

title attribute:

The title attribute defines some extra information about the element.
The value of the title attribute will be displayed as a tool-tip when the mouse hovers over the element.
