Hyperlinks

Introduction

Links are found in nearly all web pages.
Links allow users to click their way from page to page.
HTML links are called Hyperlinks.
You can click on a link and jump to another document.
The HTML a tag defines a hyperlink.
The most important attribute of the a element is the href attribute, which indicates the link's destination.

target attribute:

By default, the linked page will be displayed in the current browser window.
To change this, we must specify another target for the link.
The target attribute specifies where to open the linked document.
Values of the target attribute include:
-> _self: the default value
-> _blank: opens the document in new tab.
-> _parent: opens the document in parent frame.
-> _top: opens the document in full body of the window.

use image as a link:

Opens a link when an image is clicked.

button as a link:

Opens a link when an button is clicked.
To use an HTML button as a link, we have to add some JavaScript code.
JavaScript allows us to specify what happens at certain events, such as a button click.

link to an email address:

Allows us to send email to linked email address on click.
